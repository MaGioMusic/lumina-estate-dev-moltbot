## 📝 **რა არის CURSOR RULES?**
Cursor Rules არის ინსტრუქციები, რომლებიც ეუბნება AI-ს როგორ იმუშაოს თქვენს პროექტზე. ეს ისეთივეა, როგორც ახალ თანამშრომელს ვუხსნით კომპანიის წესებს.

## 🔧 **USER RULES (მომხმარებლის წესები)**
*ეს წესები მუდმივად მოქმედებს ყველა პროექტზე*

### **1. კოდის წერის სტილი:**
```
- ყოველთვის იყენე TypeScript (JavaScript-ის უსაფრთხო ვერსია)
- იყენე ფუნქციური კომპონენტები (არა class-ები)
- სტილებისთვის მხოლოდ Tailwind CSS
- ყოველთვის დაამატე error handling (შეცდომების დამუშავება)
```

**რას ნიშნავს:**
- **TypeScript** - კოდი უფრო ნათელი და შეცდომების გარეშე
- **ფუნქციური კომპონენტები** - მოდერნული React-ის სტილი
- **Tailwind CSS** - სტილები HTML-ის შიგნით, ლამაზად და სწრაფად

### **2. ინტერფეისი და დიზაინი:**
```
- ყოველთვის mobile-first (ტელეფონისთვის პირველ რიგში)
- დაამატე loading states (ჩატვირთვის ინდიკატორები)
- იყენე Framer Motion ანიმაციებისთვის
- ყოველთვის ARIA labels (ხელმისაწვდომობისთვის)
```

**რას ნიშნავს:**
- **Mobile-first** - ჯერ ტელეფონზე კარგად უნდა გამოიყურებოდეს
- **Loading states** - როცა რაღაც იტვირთება, მომხმარებელმა უნდა იცოდეს
- **ARIA labels** - ბრმა ადამიანებიც შეძლებენ საიტის გამოყენებას

### **3. უსაფრთხოება და ოპტიმიზაცია:**
```
- ყოველთვის DOMPurify ყველა input-ისთვის
- იყენე Zod მონაცემების შესამოწმებლად
- დაამატე React.memo ძვირადღირებული კომპონენტებისთვის
- ზუსტად ოპტიმიზაცია გაუკეთე სურათებს
```

**რას ნიშნავს:**
- **DOMPurify** - ვერავინ შეძლებს ვირუსების ჩასმას
- **Zod** - შემოსული მონაცემები შეამოწმოს სწორია თუ არა
- **React.memo** - კომპონენტი არ გადაირთვება უსაფუძვლოდ

*ეს წესები მხოლოდ Lumina Estate-ისთვის*

### **1. ფაილების სახელები და სტრუქტურა:**
```

```

**მაგალითი:**

```

### **4. კომენტარები და დოკუმენტაცია:**
```
- რთული ფუნქციებისთვის JSDoc კომენტარები

## 🎯 **პრაქტიკული მაგალითები**

### **✅ კარგი კოდი:**
```typescript
interface PropertyCardProps {
  property: Property;
  onSelect: (id: string) => void;
}

export const PropertyCard: React.FC<PropertyCardProps> = ({ 
  property, 
  onSelect 
}) => {
  return (
    <motion.div
      whileHover={{ scale: 1.02 }}
      className="bg-white rounded-lg shadow-md p-4"
      onClick={() => onSelect(property.id)}
    >
      <h3 className="text-lg font-semibold">{property.title}</h3>
    </motion.div>
  );
};
```

### **❌ ცუდი კოდი:**
```typescript
// ცუდი - არ არის TypeScript, აქვს inline styles
const card = (prop) => {
  return <div style={{backgroundColor: 'white'}} onClick={() => prop.click()}>
    <h3>{prop.title}</h3>
  </div>
}
```
