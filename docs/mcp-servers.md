### MCP სერვერები — Lumina Estate

ქვემოთ არის ყველა კონფიგურირებული MCP სერვერი, მათი დანიშნულება და გამოყენების სცენარები. წყაროებია `c:\Users\User\.cursor\mcp.json` და `lumina-estate/.cursor/mcp.json`.

- **linear**: Linear-ის ოფიციალური MCP (`mcp-remote https://mcp.linear.app/sse`).
  - **თვისები**: Issues/Projects/Teams-ზე წვდომა, ამოცანების შექმნა/განახლება.
  - **როდის ვიყენებთ**: ბაგების/ფიჩერების თრექინგი, ტასკების ავტომატიზაცია.

- **github**: GitHub MCP სერვერი.
  - **Env**: `GITHUB_PERSONAL_ACCESS_TOKEN` (პერსონალური ტოკენი საჭიროა).
  - **დანიშნულება**: რეპოზიტორიების, PR-ების, issu e-ების წაკითხვა/მართვა.

- **brave-search**: Brave ძებნის MCP.
  - **Env**: `BRAVE_API_KEY`.
  - **როდის ვიყენებთ**: სწრაფი ვებ-ძიება საუკეთესო პრაქტიკებისთვის.

- **context7**: Context7 MCP (Upstash).
  - **დანიშნულება**: დოკუმენტაციის соответствება, კოდის დოკ-შემოწმება.

- **chrome-devtools**: ლოკალური Chrome DevTools MCP.
  - **Args**: `--executablePath`, `--headless=false`, `--isolated=true`.
  - **როდის ვიყენებთ**: გვერდების ინსპექტირება/პერფორმანსის ანალიზი, სკრინშოტები.

- **shadcn**: shadcn UI MCP ინტეგრაცია.
  - **დანიშნულება**: UI კომპონენტების სწრაფი დამატება/გენერირება.

- **Magic MCP**: 21st.dev Magic მექანიზმი UI გენერაციებისთვის.
  - **Args**: `@21st-dev/magic@latest` და API_KEY არგუმენტში.
  - **როდის ვიყენებთ**: დიზაინ-სისტემის შესაბამისი UI ბილდერი.

- **Ref**: Ref search MCP (HTTP endpoint ტიპი).
  - **დანიშნულება**: დოკუმენტაციის/რესურსების მოძიება ერთიანი API-ით.

- **rube**: Composio Rube MCP.
  - **დანიშნულება**: მრავალ აპში პარალელური ავტომატიზაციები, გეგმების გენერაცია.

სანიშნეები და შენიშვნები
- კონფიგები ინახება ორი ადგილზე: `c:\Users\User\.cursor\mcp.json` (გლობალური) და `lumina-estate/.cursor/mcp.json` (პროექტური).
- პირველადი ჩატვირთვისას შესაძლებელია npx ჩამოტვირთვები, გააკეთეთ Reload Window Cursor-ში.
- სენსიტიური გასაღებები (PAT/API კీలు) შეინახეთ უსაფრთხოდ; საჭიროებისას ამოიღეთ რეპოდან.
