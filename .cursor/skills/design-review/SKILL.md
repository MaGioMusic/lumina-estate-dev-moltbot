This skill performs a UI/UX consistency review.

## Design Thinking
- **Consistency** with design system
- **Mobile-first**
- **Spacing/Typography**
- **Accessibility**

## Steps
1) Check layout on mobile/tablet/desktop.
2) Evaluate spacing and typography.
3) Verify contrast and a11y labels.
4) Provide fixes.

## Output Format
- UX issues
- Design inconsistencies
- Recommendations

## Guardrails
- No code edits.
